## Gebruik van GitHub
Voor dit project is GitHub gebruikt om alle bestanden overzichtelijk op één plek te bewaren. De repository is verdeeld in aparte mappen voor data, scripts, resultaten, bronnen en data stewardship. Hierdoor zijn bestanden gemakkelijk terug te vinden en blijft de workflow overzichtelijk.

GitHub is ook gebruikt voor versiebeheer. Door regelmatig wijzigingen op te slaan met commits kon ik veranderingen in scripts en documentatie bijhouden en indien nodig teruggaan naar een eerdere versie.

Daarnaast bevat de repository een README met uitleg over het project, de gebruikte methoden en de resultaten. Samen met de gedeelde scripts en packageversies maakt dit het voor andere gebruikers mogelijk om de analyse te begrijpen en opnieuw uit te voeren. Op deze manier draagt GitHub bij aan een reproduceerbaar en transparant onderzoek.
