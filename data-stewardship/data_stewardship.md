## Data Stewardship
Databeheer is essentieel voor betrouwbaar en reproduceerbaar onderzoek. Door data, scripts en resultaten op een gestructureerde manier op te slaan, kunnen analyses eenvoudig worden gecontroleerd, herhaald en gedeeld met andere onderzoekers.

Voor dit project is een duidelijke mappenstructuur aangehouden, waarbij ruwe data, scripts, resultaten en referenties in afzonderlijke mappen zijn opgeslagen. Hierdoor blijven bestanden overzichtelijk en zijn analyses eenvoudig terug te vinden.

```text
├─ Bronnen/              # literatuur
├─ data/                 # RNA-seq data en count matrix
├─ data_stewardship/     # Documentatie over data beheer en github
├─ resultaten/           # Figuren
├─ scripts/              # R-scripts voor de analyse
└─ README.md
